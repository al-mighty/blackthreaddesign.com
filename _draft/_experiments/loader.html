Controls

Left mouse / touch: Rotate camera

Middle mouse / Pinch: zoom

Right mouse / Two finger touch: Pan camera

Currently supported formats

Three.js native JSON format: Complete scene, or a file containing and geometry or buffer geometry.

FBX: Binary higher than version 6.1 and ASCII higher than version 7.0

GLTF: .gltf or .glb file, only embedded textures supported for now, animations not yet supported.

Supported texture formats

JPG, PNG, GIF, BMP, DDS

Archive support

Models can be loaded in ZIP format, where the ZIP archive contains a single scene and associated textures. The directory structure of the zip does not matter.

Limitations of archive support:

  • Currently on support FBX
  • Only one model / scene per zip
  • All textures must have a unique name, duplicate textures will be ignored.

Model Info

Faces: Vertices:

Note that this is the number of faces / vertices as rendered by three.js, which may be different from the numbers reported in programs. This is especially true if the model has quads (4-sided polygons) or higher, as these are automatically subdivied into triangles on load.
